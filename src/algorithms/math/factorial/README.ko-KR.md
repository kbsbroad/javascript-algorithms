# Factorial

수학에서 'n!'으로 표시된 음수가 아닌 정수 'n'의 계승 값은 'n'보다 작거나 같은 모든 양의 정수의 곱입니다. 예를 들면:

```
5! = 5 * 4 * 3 * 2 * 1 = 120
```

| n     | n!                          | 
| ----- | --------------------------: |
| 0     | 1                           |
| 1     | 1                           |
| 2     | 2                           |
| 3     | 6                           |
| 4     | 24                          |
| 5     | 120                         |
| 6     | 720                         |
