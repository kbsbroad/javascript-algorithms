# 데카르트 곱

집합 이론에서 데카르트 곱은 여러 세트로부터 집합 (또는 집합 또는 단순히 곱)을 반환하는 수학 연산입니다. 즉, 집합 A와 B에 대해, 데카르트 곱 A × B는 a ∈ A와 b ∈ B인 모든 순서쌍 (a, b)의 집합이다.

Cartesian product `AxB` of two sets `A={x,y,z}` and `B={1,2,3}`
